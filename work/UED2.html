医院自助终端

时间(Time)： 2018.07-2018.08 角色(Role)： UI Designer 工具(Tools)： PS、Protopipe
项目为广东一个县级人民医院设计自助终端，方便居民进行查看个人费用，查询价格，查看医生排班

This project is a self-service terminal for a county-level people's hospital in Guangdong province,
which makes it more convenient for residents to check personal expenses, medical prices, and inquire doctors' schedules.
